## 项目简介
`my-bilibili` 是一套基于当下前沿技术的的前后端分离的**仿B站**的的项目，旨在搭建起一个高性能的弹幕视频网站，目前仅包括后台系统，主要包括 用户/视频检索模块、文件模块、用户订阅、用户中心、评论模块、视频投稿、视频播放和实时防遮挡弹幕等主要功能模块。
